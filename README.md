# 📊 Exam Score Analysis using Z-Test & t-Test

## 📌 Overview

This mini statistical analysis project demonstrates the practical implementation of hypothesis testing techniques using Python.

The project focuses on analyzing student exam score datasets using:
- p-value interpretation
- Z-Test
- Single Sample t-Test
- Independent 2-Sample t-Test
- Paired 2-Sample t-Test

The goal of this project is to understand how inferential statistics helps in making data-driven decisions using real-world scenarios.

---

# 🚀 Problem Statement

A school wants to analyze student exam performance using statistical hypothesis testing methods.

The analysis aims to determine:
- whether average student scores significantly differ from expected values,
- whether two classes perform differently,
- and whether student performance improves after training sessions.

Statistical tests are used to validate these assumptions using exam score data.

---

# 🎯 Objectives

- Understand the concept of p-value
- Interpret statistical significance
- Perform Z-Test using Python
- Implement Single Sample t-Test
- Compare two independent groups using Independent t-Test
- Analyze before-and-after performance using Paired t-Test
- Gain practical exposure to inferential statistics
